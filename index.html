Piskel Clone

This project is a Final Task from the Second stage of Rolling Scopes School Q1 2019.

It is a clone of Piskel app

Project Functionality

You can use next tools:

  • Pencil Tool
  • Paint Bucket
  • Color Picker
  • Eraser
  • Stroke Rectangle
  • Fill Rectangle
  • Select Pen Size (1 - 4 units)
  • Select primary & secondary color

And also you can use Frame management, Preview Window, Full Screen Button and export result to gif image changing canvas size, see cursor coordinates!!!
